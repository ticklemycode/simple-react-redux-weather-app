### Getting Started

#### Familiar with Git?
Checkout this repo, install dependencies, then start the gulp process with the following:

```
> git clone git@github.com:ticklemycode/simple-react-redux-weather-app.git
> cd simple-react-redux-weather-app
> npm install
> npm start
```
